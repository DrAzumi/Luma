# LUMA 🌕

Control your smart lights with room scenes, BPM sync, and song choreography.

## Requirements
- Node.js 16+
- Python 3.8+
- Smart bulbs on same WiFi (Tuya/Havells/Syska or Philips WiZ)

## Setup
git clone https://github.com/you/luma.git
cd luma
npm install
npm run setup    ← guided wizard, takes 5 minutes

## Run
npm run dev      ← opens at http://localhost:5173
